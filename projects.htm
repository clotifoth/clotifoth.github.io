RedditScraper

In this project, I built a program which reads a series of reddit URLs from a text file, then pulls the HTML content of both the main content / link and the comments section, and stores it organized by subreddit, then thread. For easy readability, the external reddit stylesheet is added as a meta reference tag in each html file.

checkersplugin

This project is a really quick test PHP project in which I learned how to create plugins for DokuWiki. It takes a checkers tag with an ASCII representation of a checkers game state, and outputs an image representing the game state.

Version Control Workshops

Version control is an incredibly important tool in software development, ensuring accountability for code changes and data integrity in case of local device failure. I created a handful of workshops staged as events hosted by my university IEEE chapter. These workshops are aimed at explaining the joys of version control using Git with remote repositories to beginner Computer Science and Computer Engineering students,. I've put two of my workshops here, public; this workshop focused on the very basics of initializing and managing a repository with a single-user workflow in mind, then this workshop expands the scope to include branch management and multi-user workflow strategy.
